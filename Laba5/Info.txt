Выхдные файлы: 	lb5 - сама программа
		fun.dll(win) - динамическая библиотека
 		fun.so(linux) - динамическая библиотека
		Allfiles(в папке Files) - файл с информацией из всех файлов
Входные файлы, которые будут читаться, находятся в папке Files.
Файл, в который будет записана информация из файлов, также находится в папке
Files с именем Allfiles.txt.
Данная реализация претендует на кроссплатформенность т.к. один main-файл,
один make-файл, один header-файл.
